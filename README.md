## Installation API 

```bash
$ cd /api
```

```bash
$ yarn install
```

## Running the DB

```bash
yarn run db:start
```

## Running the API
```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Installation Client

```bash
$ cd /client
```

```bash
$ yarn install
```

## Start Client
Develop mode
```bash
$ yarn dev
```

Prod
```bash
$ yarn build
```
